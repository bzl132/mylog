##我的博客项目一览
项目概况：这个项目将采用spring-boot的方式搭建后端，使用docker方式部署.前端使用React为基础语言，用antd + webpack构建。
###1.0版本要完成的功能

1. 用户登录/注册/及修改用户信息功能。
2. 用户发表文章/编辑文章/标记文章/删除文章/收藏文章/及文章列表等功能。
3. 对文章进行评论，点赞功能。
